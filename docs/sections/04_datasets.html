Datasets

The project uses time-series data from EarthScope/IRIS, PhysioNet, and the UCR Time Series Classification Archive to evaluate retrieval beyond a single domain. EarthScope/IRIS provides seismic waveforms with event metadata for matching similar waveform events, PhysioNet provides curated biomedical signals (ECG, EEG) with benchmark annotations for meaningful temporal events, and UCR provides standardized datasets suited for controlled ML comparisons. Across all sources, we represent data as fixed-duration windows around events or salient patterns, which become the retrieval units for preprocessing, embedding, and vector-database indexing. Links: EarthScope/IRIS, PhysioNet, UCR Archive.
